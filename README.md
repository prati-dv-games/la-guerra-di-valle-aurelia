# La Guerra di Valle Aurelia

A beat-em-up game about **Daniele "il Conte"** fighting the gang of Valle Aurelia across the streets of Prati, Rome.

## How to Play

Open `index.html` in any browser (Chrome, Firefox, Safari, Edge). No installation needed.

### Controls

| Key | Action |
|-----|--------|
| A/D or Arrow Left/Right | Move |
| W or Arrow Up | Jump |
| Z or Space | Attack |
| X | Dodge (invincibility frames) |
| 1 | Helper: Pirox (DJ stun) |
| 2 | Helper: Tommy (drunk knife) |
| 3 | Helper: Cicca the cat (heal) |
| 4 | Helper: Il Mostro (speed boost) |
| S | Open shop (at Vanni hub) |
| M | Mute/unmute music |
| ESC | Back / return to hub |

### Bosses

1. **LaFed** (Via Plava) - Vain rugby guy, raps mid-fight
2. **Gioacchino** (Down Under Pub) - Baton + sister Regina throws meat
3. **Il Dale** (Piazza Bainsizza) - Tall and dumb, hits hard
4. **Ciccio Ruver** (Pizzarium) - Mohawk pizza maker
5. **Pisellino** (Piazza Mazzini) - Final boss, moped + uncle Mariuso

### Weapons

Buy weapons at the Vanni shop between levels:
Pugni > Sanpietrino > Bastone > Bottiglia Rotta > Machete > Catena > Pistola > La Lupa (legendary)

## How to Share with Friends

### Option 1: Send the file directly (easiest)
Just send `index.html` via WhatsApp, Telegram, email, or any messaging app. Your friend opens it in their browser. Done.

### Option 2: Free hosting with Netlify Drop (shareable link)
1. Go to https://app.netlify.com/drop
2. Drag the `la-guerra-di-valle-aurelia` folder onto the page
3. Get a free URL like `https://random-name.netlify.app`
4. Send the link to anyone!

### Option 3: GitHub Pages (permanent link)
1. Create a GitHub account if you don't have one
2. Create a new repository called `la-guerra-di-valle-aurelia`
3. Upload `index.html` to the repository
4. Go to Settings > Pages > Source: main branch
5. Your game will be at `https://yourusername.github.io/la-guerra-di-valle-aurelia/`
